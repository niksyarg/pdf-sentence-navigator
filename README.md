# PDF Sentence Navigator - Chrome Extension (Manifest V3)

ეს არის Chrome Extension-ი, რომელიც საშუალებას გაძლევთ კლავიატურის მეშვეობით იმოძრაოთ Chrome-ში გახსნილი PDF დოკუმენტის წინადადებებს შორის.

## ფუნქციონალი
- `TAB` — გადასვლა მომდევნო წინადადებაზე.
- `SHIFT + TAB` — დაბრუნება წინა წინადადებაზე.
- აქტიური წინადადება ავტომატურად გამოიყოფა (Highlight) ბრაუზერის მიერ და ხდება მასზე ავტომატური სქროლინგი.
- **ლოკალურობა:** პროექტი მუშაობს 100% ლოკალურად ბრაუზერში.

## როგორ ჩავრთოთ Extension-ი
1. გახსენით `chrome://extensions/`.
2. ჩართეთ **Developer mode**.
3. დააჭირეთ **Load unpacked** და აირჩიეთ პროექტის საქაღალდე.
4. *მნიშვნელოვანია:* Extension-ის დეტალებში (Details) ჩაურთეთ **"Allow access to file URLs"**.

## როგორ შევამოწმოთ
1. გახსენით ნებისმიერი PDF ფაილი Chrome-ში.
2. დაელოდეთ 2 წამი, დააკლიკეთ მაუსით ტექსტს და გამოიყენეთ `TAB` ან `SHIFT + TAB`.
