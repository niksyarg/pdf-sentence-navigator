# PDF Sentence Navigator - Chrome Extension

ეს არის Chrome-ის ექსტენშენი, რომელიც საშუალებას გაძლევთ მარტივად, კლავიატურის გამოყენებით იმოძრაოთ PDF დოკუმენტის წინადადებებს შორის. ექსტენშენი ანაცვლებს ბრაუზერის სტანდარტულ PDF რიდერს და უზრუნველყოფს სწრაფ და მოხერხებულ ნავიგაციას.

## 🚀 ფუნქციონალი
* **კლავიატურით ნავიგაცია:**
  * `TAB` — გადასვლა მომდევნო წინადადებაზე.
  * `SHIFT + TAB` — დაბრუნება წინა წინადადებაზე.
* **ავტომატური სქროლინგი:** აქტიური წინადადება ავტომატურად ხდება Highlight-ირებული და ბრაუზერი ავტომატურად ასქროლავს გვერდს მისკენ.
* **მრავალგვერდიანი მხარდაჭერა:** ექსტენშენი ავტომატურად ტვირთავს PDF-ის ყველა გვერდს.
* **100% ლოკალურობა:** ფაილების დამუშავება ხდება სრულად ბრაუზერში, თქვენს კომპიუტერში.

## 🛠️ ინსტალაცია
1. გადმოწერეთ პროექტის ფაილები თქვენს კომპიუტერში.
2. გახსენით Chrome-ში მისამართი: `chrome://extensions/`.
3. ჩართეთ **Developer mode** (მარჯვენა ზედა კუთხეში).
4. დააჭირეთ **Load unpacked** და აირჩიეთ პროექტის საქაღალდე.
5. **მნიშვნელოვანი:** ექსტენშენის დეტალებში (Details) ჩაურთეთ ოფცია **"Allow access to file URLs"**, რათა ექსტენშენმა შეძლოს თქვენი კომპიუტერიდან PDF ფაილების გახსნა.

## 📖 როგორ გამოვიყენოთ
1. **ავტომატური რეჟიმი:** უბრალოდ გახსენით ნებისმიერი PDF ფაილი ბრაუზერში. ექსტენშენი ავტომატურად გადაამისამართებს მას თავის ვიუერში.
2. **მექანიკური რეჟიმი:** გახსენით ექსტენშენის ინტერფეისი და ატვირთეთ ფაილი `ატვირთე PDF ლოკალურად` ღილაკით.
3. **ნავიგაცია:** მას შემდეგ, რაც PDF სრულად ჩაიტვირთება, გამოიყენეთ `TAB` და `SHIFT + TAB` წინადადებებს შორის გადასაადგილებლად.

## ⚙️ ტექნოლოგიური სტეკი
* **PDF.js:** PDF ფაილების გასახსნელად და რენდერისთვის.
* **Manifest V3:** Chrome ექსტენშენების თანამედროვე სტანდარტი.
* **JavaScript (ES6 Modules):** სუფთა და მოდულარული კოდის სტრუქტურისთვის.
